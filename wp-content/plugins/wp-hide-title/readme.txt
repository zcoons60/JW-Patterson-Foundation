Yet Another Plugin that Allows users to Hide Post/ Page Title.


== Description ==
This plugin adds a simple checkbox on page or post creating page. 
No Effect on site load time. 
This is just a simple tiny plugin to hide any post or page title.

== Installation ==
This section describes how to install the plugin and get it working.

Search And install from your wordpress admin dashboard.
or
1. Unzip and Upload the wp-hide-title folder to your plugin directory.
2. Activate it. 

== Frequently Asked Questions ==
How to configure this plugin?
Just simply install and activate this plugin and you will see a extra box to hide title when adding a new page or post.
